# Interval Timer
This is an interval timer with 4 total workouts:
  - Beginner Workout A
  - Advanced Workout A
  - Beginner Workout B
  - Advanced Workout B

Here's a [video](https://youtu.be/vc1E5CfRfos) showing how to do each exercise.

Instructions:
  - Download the repository
  - cd to the repository folder
  - type `make all`
  - type `./workout`
  - Enjoy.
